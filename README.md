
# Calculadora de IMC

Aplicativo de calcular IMC.


## Funcionalidades

- Calcular o IMC, Validador de inpus.
- Multiplataforma


## Aprendizados

Conceitos importantes de Dart e flutter, usar Widget, Criar validações de inputs, usar GlobalKey<FormState>.
